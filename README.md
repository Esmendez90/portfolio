## Portfolio

### 1. [The Berry Homeland](https://github.com/Esmendez90/berryhomeland)
- **Description**: A comprehensive platform designed to showcase the expertise of an agricultural business specializing in Abaca fiber manufacturing. The site provides detailed information about the company's history, products, and practices, featuring a clean, user-friendly interface.
- **Technologies**: Bootstrap, HTML, CSS, JavaScript
- **Features**: Company history, product details, contact information, gallery page.

### 2. [JCB Real Estate Services](https://github.com/Esmendez90/jcbhome)
- **Description**: A platform providing users with comprehensive information about a real estate agent, their professional journey, and team members, facilitating client connection with the agent's expertise and experience.
- **Technologies**: React, HTML, CSS, JavaScript, Photoshop
- **Features**: Agent background, career insights, team profiles.

### 3. [Spark Events](https://github.com/Esmendez90/spark)
- **Description**: A dynamic application leveraging the Ticketmaster API to enable users to search for a wide range of events, including concerts, attractions, and sports events.
- **Technologies**: HTML, CSS, JavaScript
- **Features**: Event search, ticketing options, social media links.

### 4. [Gullible](https://github.com/Esmendez90/gullible)
- **Description**: An application allowing users to search for detailed information about movies and series, leveraging the OMDB API for comprehensive data access.
- **Technologies**: HTML, CSS, JavaScript
- **Features**: Movie and series search, detailed information display.

### 5. [Weather Dashboard](https://github.com/Esmendez90/weather-dashboard)
- **Description**: Provides real-time weather forecasts for any city, including a detailed 5-day outlook, with customized and animated weather icons.
- **Technologies**: HTML, CSS, JavaScript
- **Features**: Real-time weather data, animated icons.
